# Todo List (Vue 3 + Vite)

A simple, stylish todo list built with Vue 3 and Vite. Add, edit, mark complete, filter, and clear tasks with local storage persistence.

## Getting started

1. Install dependencies (requires Node 18+):

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

4. Preview the production build:

   ```bash
   npm run preview
   ```
